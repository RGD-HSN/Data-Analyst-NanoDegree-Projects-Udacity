# Ford GoBike Data Analysis & Visualization

## Overview

This project explores the Ford GoBike bike-sharing dataset from the San Francisco Bay Area. The goal is to analyze trip patterns, user behavior, and key factors that influence how the service is used.

The project is divided into two parts:

* **Exploratory Data Analysis (EDA)**
* **Explanatory Visualization (Presentation)**

---

## Dataset

The dataset contains information about individual bike trips, including:

* Trip duration
* Start and end times
* Station names and locations
* User type (Customer or Subscriber)
* User demographics (age and gender)

---

## Objectives

* Understand the distribution of trip durations
* Analyze differences between user types
* Explore how time and demographics affect usage patterns
* Identify key behavioral trends in bike-sharing usage

---

## Key Insights

* Most trips are short (typically under 30 minutes)
* Subscribers tend to take shorter, more frequent trips (likely commuting)
* Customers take longer and more irregular trips
* Usage patterns vary across days of the week and time of day

---

## Tools & Libraries

* Python

---

## Conclusion

This analysis provides insights into how users interact with bike-sharing systems and highlights differences in behavior between casual users and subscribers.
