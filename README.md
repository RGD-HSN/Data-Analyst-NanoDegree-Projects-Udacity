2. **Real-World Data Wrangling (Google Play Store Data)**

   * Focuses on cleaning, merging, and analyzing app data and user reviews to study sentiment.

3. **Exploratory Data Analysis & Visualization (Ford GoBike Dataset)**

   * Analyzes bike-sharing trip data to uncover patterns in user behavior and trip characteristics.
